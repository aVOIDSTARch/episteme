# Sample topic file

This folder contains a single sample topic file for the Ideas System so you can see the format without creating `~/ideas/` first.

- **[getting-started.md](getting-started.md)** — Example topic with two entries. Structure matches [_idea-file-schema.md](../_idea-file-schema.md).

To use the Ideas System for real: create `~/ideas/` (or set `IDEAS_ROOT`), then add topic files there. You can copy `getting-started.md` into that folder as a first topic.
