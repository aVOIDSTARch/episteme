## Conventions

- **Leading underscore (_)** — A file or folder name starting with `_` is a schema or convention doc (e.g. `_project-schema.md`, `_idea-file-schema.md`). It defines structure and rules; it does not describe skill behavior. Skill behavior is in SKILL.md files or notes.
- **Paths** — All links above are relative to `episteme-framework/`. From repo root, prefix with `episteme-framework/`.
